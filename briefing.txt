PROJECT OVERVIEW
----------------
Name: LuxeDrive
Type: Car dealership management system (gestion de voitures)

DATABASE
--------
File: database.db (located in the root project folder, 84 Ko)
Full path: C:\Users\PC\Desktop\imad docc projet fin d'anner\gestion-voiture\database.db
Engine: SQLite
Viewer tool: DB Browser for SQLite (sqlitebrowser.org) — open database.db to browse/edit data
  OR: SQLite Viewer extension in VS Code

The app writes directly to this file. Any car/client/sale added through the website
is saved here instantly.


DATABASE SCHEMA (tables)
------------------------
- marque          → car brands
- modele          → car models (linked to marque)
- voiture         → cars (VIN, model, type, plate, year, buy price, sell price, status)
- client          → customers
- fournisseur     → suppliers
- agent           → sales agents (with commission rate %)
- commande_achat  → purchase orders (from suppliers)
- ligne_achat     → cars per purchase order (links commande_achat → voiture by VIN)
- vente           → sales transactions
- ligne_vente     → cars per sale (links vente → voiture by VIN)
- facture         → invoices
- paiement        → payments
- historique_proprietaire → ownership history
- user            → app users (roles: admin / user)

User roles:
  admin → full access to everything
  user  → can only browse cars and buy them


APP PAGES (routes in app.py)
-----------------------------
/                   → Dashboard (stats: cars, sales, clients)
/voitures           → Car inventory list
/voitures/add       → Add a new car
/voitures/buy/<vin> → Buy a car (user role only)
/clients            → Client management (CRUD)
/agents             → Agent management (CRUD)
/fournisseurs       → Supplier management (CRUD)
/marques            → Brand management + model management
/marque/<id>        → Brand detail page (models for that brand)
/transactions       → Sales & purchases history
/ventes/add         → Record a new sale
/achats/add         → Record a new purchase (ADDED July 2026)
/devis/select       → Select car for quote
/devis/<vin>/<id>   → Generate invoice/quote (PDF-ready)
/confirmation/<id>  → Bon de confirmation for a sale
/login              → Login page
/signup             → Signup page
